# Défi Turing – Solutions en Python

Ce dépôt regroupe mes solutions aux problèmes du **Turing** proposés sur la plateforme :  
https://apprendre-en-ligne.net/turing/

L’objectif est de résoudre progressivement ces exercices en **Python**, en me concentrant d’abord sur la clarté et la compréhension des algorithmes.
